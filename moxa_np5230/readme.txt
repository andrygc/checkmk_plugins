DESCRIPCION:
- Este plugin de Checkmk está diseñado para monitorear dispositivos Moxa NP5230 utilizando SNMP. Procesa y presenta valores de MIBs específicas en un formato legible para su monitoreo y administración. Crea un servicio con cada valor de MIB que se establezca.

INSTALACION:
1. colocar los archivos manualmente en su servidor
2. acceder como "omd su" en su sitio
3. escribir el comando "mkp add /ruta/del/archivo/moxa_np5230-1.0.0.mkp"
4. escribir el comando "mkp enable moxa_np5230"
5. escribir el comando "mkp list" para ver el listado de todas las extensiones.

AYUDA:
- visite https://docs.checkmk.com/latest/en/mkps.html para mas informacion.
